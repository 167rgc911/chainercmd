# ChainerCMD

You can write all configuration of training in a YAML file, and start training with one line:

```
$ chainer train config.yml
```

## Installation

```
pip install chainercmd
```

## Requirement

- Python 3.4.4+
- Chainer 2.0.1+
- PyYAML 3.12+

## Create new project

```
$ chainer init
```
